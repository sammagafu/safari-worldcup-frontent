# Hero background images

Add these files for the Home hero section:

- **bg-mobo.jpg** – used on mobile (shown below `md` breakpoint)
- **bg-web.jpg** – used on desktop (shown from `md` breakpoint up)

Supported formats: `.jpg`, `.jpeg`, `.webp`, or `.png`. If you use a different extension, update the imports in `src/views/HomeView.vue`.
